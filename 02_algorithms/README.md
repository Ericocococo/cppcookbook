# 02_algorithms — 算法与设计模式

手写数据结构、经典算法、Modern C++ 设计模式。

| 目录 | 内容 |
|------|------|
| 01_data_structures | 链表、树、图、堆、并查集、跳表、布隆过滤器 |
| 02_classic | 排序、搜索、DP、贪心、回溯、分治 |
| 03_patterns | GoF 设计模式在 Modern C++ 中的实现 |

## 术语速查

| 术语 | 含义 |
|------|------|
| DP | Dynamic Programming — 动态规划 |
| GoF | Gang of Four — 设计模式经典著作《Design Patterns》的四位作者 |
| CRTP | Curiously Recurring Template Pattern — C++ 静态多态的惯用法，见 [01_language](../01_language/README.md) |
